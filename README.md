ROCKETSEAT
Exercício 01 do Stage 4 

Nesse desafio foi solicitado a todos que fosse feito o calculo de dois números, com isso foram feitos a:

[] Soma.
[] Subtração.
[] Multiplicação.
[] Divisão.
[] O resto da divisão.
[] Se a soma era par ou impar.
[] Se os dois numeros eram iguais ou diferentes.
